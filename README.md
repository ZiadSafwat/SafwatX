
# 💻SafwatX✨

**SafwatX** is a network management tool inspired by popular tools like **TuxCut** and **EvilLimiter**. It provides an intuitive interface to scan, block, and unblock devices on your local network. previously this [app](https://github.com/ZiadSafwat/SafwatX/tree/old-version(backend%2Bdesktop)) were built with **Flutter** and **Python (Flask)** and works on linux only now i am building it to work on both windows and linux so keep in touch i have built [cli tool](https://github.com/ZiadSafwat/Windows-ARP-CLI-Tool) to help me port app to windows .
